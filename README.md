the-wrath-of-kotlin
===================

Random code with the Kotlin programming language

    git clone https://github.com/CarloMicieli/the-wrath-of-kotlin
    
To generate the Kdoc (the Kotlin documentation):

    ./gradlew dokka

To run the test suite:

    ./gradlew test

Requirements
------------

* Java Development Kit (8.0+)
